AltAllele¶

class hail.representation.AltAllele(ref, alt)[source]¶

An object that represents an allele in a polymorphism deviating from the reference allele.

Parameters:
  • ref (str) – reference allele
  • alt (str) – alternate allele

is_transition()[source]¶

True if this alternate allele is a transition SNP.

This is true if the reference and alternate bases are both purine (A/G) or both pyrimidine (C/T). This method raises an exception if the polymorphism is not a SNP.

Return type:bool
is_transversion()[source]¶

True if this alternate allele is a transversion SNP.

This is true if the reference and alternate bases contain one purine (A/G) and one pyrimidine (C/T). This method raises an exception if the polymorphism is not a SNP.

Return type:bool
