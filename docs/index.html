Add a new annotation

To add a new annotation to the database:

  1. Format your data into a flat text file (.tsv, .csv, .txt, etc).

    Each row should contain a column with a unique ID in one of the following formats:

    • Variant - "CHR:BP:REF:ALT" (e.g. "1:10505:A:T")
    • Interval - "CHR:START-END", where START is inclusive and END is exclusive (e.g. "1:10000-15000")

  2. Use bgzip to compress the text file into .bgz format.
  3. Upload the text file to gs://annotationdb-submit/data/.

  4. Enter the title and Hail path of your new annotation in this form.

    • The title can be any text.
    • The annotation path must begin with "va.".

  5. After submitting this form, edit the documentation with details about your data and the variables it contains.
